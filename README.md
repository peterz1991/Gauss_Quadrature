# Gauss_Quadrature
Demos of various Gauss Quadrature rules 

This demo includes the implementation of the following Gauss Quadrature (GQ) rules:

- GQ with 
  - Chebyshev polynomial of first kind;
  - Chebyshev polynomial of second kind;
  - Legendre polynomial;
  - Laguerre polynomial;

- Gauss-Radau rules
- Gauss-Lobatto rules
- Anti-Gauss rules
